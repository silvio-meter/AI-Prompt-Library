# Prompt Examples & Improvements

## Example 1: Before vs After

**Before (weak):**
"Write an Instagram caption about wellness."

**After (strong):**
"You are an expert content creator for a premium wellness brand. Write an engaging Instagram caption (max 140 words) about [TOPIC]. Tone: friendly, trustworthy, slightly inspirational. Include 3–5 relevant emojis and end with a clear call-to-action."

## Key Improvements
- Added role and context
- Defined constraints (length, tone, structure)
- Included placeholders for customization
